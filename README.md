This is a simple Python tool that helps you do important money calculations. It is specifically designed to figure out NPV (Net Present Value) and IRR (Internal Rate of Return). 

These two tools are used to decide if a business project or investment is worth the money. You just need to put in your expected cash flow numbers and the interest rate, and the program will give you the answer. It is built to be fast, accurate, and easy for anyone to use.
